Adds a clickable background to a wordpress blog

== Description ==
A great way to monetize your blog by adding an advertisement in the background.

The plugin allows you to add a clickable background to any narrow or boxed WordPress theme. You have the option to select what image goes to the left, what image goes to the right and their links.

You can also adjust the image margin, height and z-index. 

There is also an option to make the image sticky. 

== Installation ==

Just follow the standard [WordPress plugin installation procedure](http://codex.wordpress.org/Managing_Plugins).

== Frequently asked questions ==

= I get a white background instead of the selected images=

You need to remove the themes background image via the themes settings. If there is an image set in the theme, a white background is displayed.
